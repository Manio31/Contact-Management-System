# Contact-Management-System
README: Contact Management System (Java Swing)This repository contains a desktop-based Contact Management System built with Java Swing. It provides a user-friendly interface to store, view, search, and manage personal contacts using a tabular layout.### Project FeaturesTable-Based View: Uses JTable and DefaultTableModel to display contacts in a clean, organized grid.Dynamic Storage: Utilizes an ArrayList to manage contact objects in the backend, ensuring data consistency with the UI.Smart Search: Includes a search feature that automatically scans the list and highlights the matching contact row.Easy Management: Allows users to add contacts via text fields and delete them by simply selecting a row in the table.Error Handling: Features built-in validation to ensure fields aren't empty and uses JOptionPane for user alerts.### How to Run the Program1. PrerequisitesJDK 8 or higher installed on your machine.An IDE (IntelliJ, Eclipse, VS Code) or the command line.2. CompilationOpen your terminal/command prompt and compile the file:Bashjavac ContactManagerGUI.java
3. ExecutionRun the application:Bashjava contact.management.system.ContactManagerGUI
### System ComponentsComponentFunctionBorderLayoutManages the primary layout (Inputs at Top, Table in Center, Actions at Bottom).ArrayList<Contact>Acts as the data source for the application.DefaultTableModelControls the data displayed in the visible table.Lambda ExpressionsUsed for modern, concise button click handling (addActionListener).### User GuideAdding: Enter a name and phone number in the top fields and click "Add Contact".Searching: Type a name in the search bar at the bottom and click "Search". The system will highlight the first matching row.Deleting: Click on any row in the table to select it, then click "Delete Selected" to remove it from the system.### Possible EnhancementsFile Persistence: Modify the code to save and load contacts from a .txt or .csv file so data isn't lost when closing the app.Edit Feature: Add a button to update existing contact details directly in the table.Sorting: Implement functionality to sort contacts alphabetically by name.
